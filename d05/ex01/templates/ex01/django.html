Ex01: Django, Framework Web

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. It was created in 2003 by Adrian Holovaty and Simon Willison as a way to build web applications quickly with less code. Django follows the "batteries-included" philosophy, providing many features out of the box, such as an ORM, authentication, and an admin interface.
